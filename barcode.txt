# Welcome

This is a premium white-label barcode scanner Android app.

## Quick Setup

### 1. Open Project
- Open Android Studio
- File > Open > Select `barcode-reader` folder
- Wait for Gradle sync

### 2. Change Branding
Edit `app/src/main/java/com/barcodereader/config/BrandingConfig.kt`:

```kotlin
object BrandingConfig {
    const val APP_NAME = "Your App Name"
    const val COMPANY_NAME = "Your Company"
    const val SUPPORT_EMAIL = "support@you.com"
    const val PRIVACY_POLICY_URL = "https://yoursite.com/privacy"
}
```

### 3. Change Colors
Edit the same file:

```kotlin
const val PRIMARY_COLOR = 0xFF6750A4L
const val SECONDARY_COLOR = 0xFF625B71L
```

### 4. Replace Icons
- Replace files in `res/mipmap-hdpi/` through `res/mipmap-xxxhdpi/`
- Keep same filenames

### 5. Build APK
- Build > Generate Signed Bundle/APK
- Choose APK or AAB
- Follow wizard

### 6. Publish
- Go to play.google.com/console
- Create app listing
- Upload APK/AAB
- Submit for review
